Key Design Decisions

Multi-Project

Central Hub manages multiple projects via registered directory paths. Each project has its own .hub/ config folder.

Storage Abstraction

Unified interface for file operations. Local mode uses Node.js fs, SaaS mode uses File System Access API.

File-First Data

All data stored as JSON/Markdown files in project directories. No database required. Git-friendly.

Agent Skill

Claude Code skill enables AI agents to create/read/update issues, docs, and planning data via file operations.
